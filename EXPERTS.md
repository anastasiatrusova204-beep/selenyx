### 10. `tg.CloudStorage` вместо `localStorage` для знака пользователя
`localStorage` очищается при обновлениях Telegram клиента на iOS. `tg.CloudStorage` синхронизируется между устройствами и надёжнее.

```javascript
// Сохранение
tg.CloudStorage?.setItem('userSign', sign, () => {});
// Чтение
tg.CloudStorage?.getItem('userSign', (err, val) => { if (val) userSign = val; });
```

Нужен fallback на `localStorage` если `CloudStorage` недоступен.

**Файл:** [app.js](app.js) — `initSplash()`, `finishOnboarding()`, `saveSettings()`

---

### 11. `viewportChanged` для iOS клавиатуры
На iOS при открытии клавиатуры (форма натальной карты) viewport меняется и layout ломается.

```javascript
tg.onEvent('viewportChanged', ({ isStateStable }) => {
  if (isStateStable) document.body.style.height = tg.viewportStableHeight + 'px';
});
```

**Файл:** [app.js](app.js) — после `tg.expand()`

### 15. Комиссия Telegram Stars: 30% с каждой транзакции

Telegram Stars — не «бесплатный» канал монетизации. С каждой Stars-оплаты Telegram забирает 30% — столько же, сколько App Store и Google Play. Разница: у Telegram нет альтернативного IAP-метода в обход.

**Что учитывать:** При цене натальной карты 690₽ пользователь платит Stars, ты получаешь ~480₽ в Toncoin. Закладывать в unit economics с первого дня.

**Файл:** Пока монетизация не реализована — держать в голове при Шаге 12.

---

### 16. Вывод средств: только Fragment → TON → рубли

Звёзды конвертируются в Toncoin через Fragment. Прямой вывод в рубли или карту — невозможен. Цепочка: Stars → Toncoin (Fragment) → биржа/P2P → рубли.

**Риск:** волатильность TON в момент конвертации + комиссии на каждом шаге. При малых оборотах (MVP) это несущественно; при 100+ платежах в месяц — уже операционная нагрузка.

**Что сделать сейчас:** открыть Fragment-аккаунт заранее, до запуска монетизации, чтобы верификация не тормозила старт продаж.

---

### 17. Платформенный риск: Telegram может изменить правила

Selenyx живёт внутри чужой платформы. Telegram может:
- поднять комиссию Stars выше 30%
- ограничить Mini App API (прецеденты были в 2023–2024)
- изменить условия вывода средств
- заблокировать бота без объяснений (нарушение ToS — даже случайное)

**Что делать прямо сейчас (до MVP):**
1. Собирать email-базу через бота (`/start` → попросить email за бонус)
2. Дублировать ключевой контент в канале Telegram (независимо от бота)
3. Не строить весь пользовательский опыт только внутри Mini App — иметь fallback на бота

**Приоритет:** Не блокирует MVP, но важно помнить при масштабировании.
