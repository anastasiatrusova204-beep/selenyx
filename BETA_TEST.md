## Как начать (новый пользователь)

1. Открыть @Selenyx_mybot в Telegram
2. Нажать **Начать** или `/start`
3. Нажать синюю кнопку **«Selenyx»** в меню чата
4. Пройти онбординг (4 шага): цель → состояние → дата рождения → знак
5. Готово — ты в приложении

## Сценарии для проверки

### Блок 1 — Онбординг

- [ ] Онбординг открывается, слайды анимированы
- [ ] Кнопка **«←»** на слайдах 2–4 возвращает назад (и кнопка Назад Telegram)
- [ ] Кнопка **«Пропустить»** работает — переход к выбору знака
- [ ] Выбор цели подсвечивает карточку
- [ ] Ввод даты рождения → автоматически определяется знак («Твой знак — Лев ♌»)
- [ ] После финиша — Telegram показывает всплывающее сообщение
- [ ] Открывается главный экран с прогнозом

### Блок 2 — Вкладка «День»

- [ ] Дата + фаза Луны + знак отображаются в шапке
- [ ] Карточка прогноза: заголовок «🌙 [фаза] · [Знак]» + текст + «Избегай»
- [ ] Домены-аккордеон: Здоровье / Работа / Отношения / Психология — тап разворачивает
- [ ] Приоритетный домен (по цели онбординга) первый и выделен меткой «★ фокус»
- [ ] В раскрытом тексте термины **подчёркнуты точками** → тап → подсказка
- [ ] Карточки «Цвет дня» и «Число дня» — тап разворачивает детали (аккордеон)
- [ ] Блок практики лунного дня: символ + название + что делать
- [ ] Кнопки **«🎯 В точку»** / **«↩ Не про меня»** работают
- [ ] Кнопка **«Поделиться»** → шаринг в Telegram
- [ ] Кнопка **«📚 База знаний»** внизу → список статей

### Блок 3 — Вкладка «Луна»

- [ ] Арка лунного цикла: точка анимированно появляется на нужной позиции
- [ ] CSS-диск Луны отображается с правильной фазой (не эмодзи)
- [ ] Фаза, знак, лунный день, градус заполнены
- [ ] Текст энергии Луны в знаке
- [ ] Чипы действий: зелёные «хорошо» и красные «осторожно»

### Блок 4 — Вкладка «Месяц»

- [ ] Сетка месяца с эмодзи фаз Луны
- [ ] Кнопки ‹ / › переключают месяц
- [ ] Тап на день → шторка с фазой и ползунком самочувствия (1–10)
- [ ] После сохранения — ячейка меняет цвет
- [ ] Кнопка «График» → bar chart

### Блок 5 — Вкладка «Оракул»

- [ ] Показывается иконка печенья с лёгким покачиванием
- [ ] Тап → взрыв частиц + текст появляется по словам (не мгновенно)
- [ ] Текст оракула личный и острый (не шаблонный гороскоп)
- [ ] Завтра текст будет другим (не повторяется каждый день)
- [ ] Кнопка «Поделиться» появляется через ~1 сек после текста

### Блок 6 — База знаний

- [ ] Кнопка «📚 База знаний» → список статей
- [ ] Тап на статью → читается полный текст
- [ ] Кнопка «Назад» Telegram → возврат к списку (не закрывает оверлей!)
- [ ] Кнопки «Назад» / «Далее» навигируют между статьями

### Блок 7 — Настройки

- [ ] Кнопка ⚙️ → настройки
- [ ] Можно сменить знак → сохраняется
- [ ] Можно изменить цель → меняет приоритетный домен на «День»
- [ ] В меню бота кнопка «🔔 Уведомления» → выбор времени

### Блок 8 — Повторный визит

- [ ] Закрыть → открыть снова: онбординг не показывается
- [ ] Прогноз загружается без задержки (кэш)
- [ ] Данные сохранились: знак, цель

Что произошло: ...
