## Что тестируем

Selenyx — Telegram Mini App с ежедневными астрологическими подсказками.
Реальные астрономические данные (Swiss Ephemeris / NASA). Без предсказаний будущего.

**Приложение:** https://t.me/Selenyx_mybot/app

---

## Как начать (новый пользователь)

1. Открыть @Selenyx_mybot в Telegram
2. Нажать **Начать** или отправить `/start`
3. Нажать кнопку **«Начать →»**
4. Пройти онбординг (3 шага) → выбрать знак зодиака
5. Готово — ты в приложении

---

## Сценарии для проверки

### Блок 1 — Первый запуск
- [ ] Онбординг открывается корректно
- [ ] Кнопка **«Пропустить»** работает (переход сразу к выбору знака)
- [ ] После выбора знака появляется подтверждение «Знак сохранён ✓»
- [ ] После закрытия подтверждения — открывается главный экран с прогнозом

### Блок 2 — Вкладка «Мой день»
- [ ] Показывается фаза Луны и знак Луны
- [ ] Карточки «Хорошо / Избегай» заполнены текстом
- [ ] Кнопки доменов (Здоровье / Работа / Отношения / Психология) открывают шторку с текстом
- [ ] В тексте шторки незнакомые термины **подчёркнуты точками** → тап → появляется подсказка
- [ ] Карточка «Цвет дня» → тап → шторка с объяснением
- [ ] Карточка «Число дня» → тап → шторка с объяснением
- [ ] Через ~3 секунды появляется баннер «Получай каждое утро 📲» (только первый раз)

### Блок 3 — Вкладка «Луна»
- [ ] Показываются фаза, знак, лунный день
- [ ] Текст энергии заполнен
- [ ] Незнакомые термины подчёркнуты → тап → подсказка

### Блок 4 — Вкладка «Месяц»
- [ ] Сетка текущего месяца с эмодзи фаз Луны
- [ ] Кнопки ‹ / › переключают месяц
- [ ] Тап на день → шторка с фазой Луны этого дня
- [ ] В шторке можно поставить оценку самочувствия (ползунок 1–10) и сохранить
- [ ] После сохранения — ячейка в сетке меняет цвет
- [ ] Кнопка «График» → bar chart по дням

### Блок 5 — Вкладка «Оракул»
- [ ] Показывается иконка печенья
- [ ] Тап → «разбивается» → появляется предсказание дня
- [ ] Кнопка «Поделиться» → открывается шаринг в Telegram
- [ ] Термины в тексте подчёркнуты → тап → подсказка

### Блок 6 — База знаний
- [ ] В меню бота есть кнопка «📚 База знаний»
- [ ] Нажатие открывает приложение со списком статей
- [ ] Тап на статью → читается полный текст
- [ ] Кнопка ☰ → возврат к списку
- [ ] Кнопки «Назад» / «Далее» навигируют между статьями

### Блок 7 — Настройки и уведомления
- [ ] Кнопка ⚙️ в правом верхнем углу → открываются настройки
- [ ] Можно сменить знак зодиака → сохраняется
- [ ] Можно выбрать время уведомлений (07:00 — 11:00)
- [ ] В меню бота кнопка «🔔 Уведомления» → показывает текущее время

### Блок 8 — Повторный визит
- [ ] Закрыть приложение → открыть снова
- [ ] Онбординг **не** показывается повторно
- [ ] Прогноз загружается без задержки (кэш)

## Как сообщить об ошибке

Прямо в боте: `/feedback` или кнопка «📝 Написать отзыв» в разделе «ℹ️ О боте».

**Формат сообщения:**
```
Устройство: iPhone 14 / Android Samsung S23 / десктоп
Вкладка: Мой день / Луна / Месяц / Оракул
Что делал: [описание действия]
Что произошло: [описание ошибки]
```
