## Что тестируем

Selenyx — Telegram Mini App с ежедневными астрологическими подсказками.
Реальные астрономические данные (Swiss Ephemeris / NASA). Без предсказаний будущего.

**Приложение:** https://t.me/Selenyx_mybot/app

---

## Как начать (новый пользователь)

1. Открыть @Selenyx_mybot в Telegram
2. Нажать **Начать** или отправить `/start`
3. Нажать синюю кнопку **«Selenyx»** в меню чата
4. Пройти онбординг (5 шагов):
   - Выбор цели (Отношения / Карьера / Самопознание / Здоровье)
   - Текущее состояние (чипы: тревога, усталость...)
   - Дата рождения → знак определяется автоматически
   - Выбор знака вручную (если нужно поменять)
5. Готово — ты в приложении

## Сценарии для проверки

### Блок 1 — Онбординг

- [ ] Онбординг открывается корректно
- [ ] Кнопка **«←»** появляется на слайдах 2–5 и возвращает назад
- [ ] Кнопка **«Пропустить»** работает (переход к последнему шагу)
- [ ] Выбор цели подсвечивает карточку
- [ ] Выбор состояния из чипов (можно тапнуть один)
- [ ] Ввод даты рождения → появляется «Твой знак — Лев ♌» автоматически
- [ ] После финиша онбординга — Telegram показывает всплывающее сообщение
- [ ] После закрытия — открывается главный экран с прогнозом

### Блок 2 — Вкладка «День»

- [ ] В шапке отображается фаза Луны и строка «❤️ Фокус: Отношения» (по выбранной цели)
- [ ] Карточка прогноза заполнена: «Сегодня хорошо» + «Избегай»
- [ ] Домены (Здоровье / Работа / Отношения / Психология) — аккордеон: тап → разворачивается
- [ ] Первый домен соответствует выбранной цели
- [ ] В раскрытом тексте незнакомые термины **подчёркнуты точками** → тап → подсказка
- [ ] Карточка «Цвет дня» → тап → шторка с объяснением
- [ ] Карточка «Число дня» — видно превью подсказки; тап → шторка с полным объяснением
- [ ] Блок «практика дня» под доменами — название лунного дня + что делать сегодня
- [ ] Через ~3 секунды при первом визите — баннер «Получай каждое утро 📲»
- [ ] Кнопки **«🎯 В точку»** / **«↩ Не про меня»** — хорошо видны, тап сохраняет реакцию
- [ ] Кнопка **«Поделиться»** → открывается шаринг в Telegram
- [ ] Кнопка **«📚 База знаний»** внизу → открывает список статей

### Блок 3 — Вкладка «Луна»

- [ ] Показываются: фаза, знак Луны, лунный день, градус
- [ ] Арка лунного цикла (визуальная дуга прогресса) видна
- [ ] Персональное послание по знаку + фазе отображается
- [ ] Чипы действий: зелёные «хорошо» и красные «осторожно»
- [ ] Незнакомые термины подчёркнуты → тап → подсказка

### Блок 4 — Вкладка «Месяц»

- [ ] Сетка текущего месяца с эмодзи фаз Луны
- [ ] Кнопки ‹ / › переключают месяц
- [ ] Тап на день → шторка с фазой Луны этого дня
- [ ] В шторке можно поставить оценку самочувствия (ползунок 1–10) и сохранить
- [ ] После сохранения — ячейка в сетке меняет цвет
- [ ] Кнопка «График» → bar chart по дням

### Блок 5 — Вкладка «Оракул»

- [ ] Показывается иконка печенья
- [ ] Тап → «разбивается» → появляется предсказание дня
- [ ] Кнопка «Поделиться» → открывается шаринг в Telegram
- [ ] Термины в тексте подчёркнуты → тап → подсказка

### Блок 6 — База знаний

- [ ] В меню бота есть кнопка «📚 База знаний»
- [ ] Нажатие открывает приложение со списком статей
- [ ] Тап на статью → читается полный текст
- [ ] Кнопка ☰ → возврат к списку
- [ ] Кнопки «Назад» / «Далее» навигируют между статьями

### Блок 7 — Настройки и уведомления

- [ ] Кнопка ⚙️ в правом верхнем углу → открываются настройки
- [ ] Можно сменить знак зодиака → сохраняется
- [ ] Можно изменить фокус (цель) → отображается на вкладке День
- [ ] Можно выбрать время уведомлений (07:00 — 11:00)
- [ ] В меню бота кнопка «🔔 Уведомления» → показывает текущее время

### Блок 8 — Повторный визит

- [ ] Закрыть приложение → открыть снова
- [ ] Онбординг **не** показывается повторно
- [ ] Прогноз загружается без задержки (кэш)
- [ ] Строка «Фокус» в шапке соответствует сохранённой цели

## Как сообщить об ошибке

Прямо в боте: `/feedback` или кнопка «📝 Написать отзыв» в разделе «ℹ️ О боте».

**Формат:**
```
Устройство: iPhone 14 / Android Samsung S23
Блок: Онбординг / День / Луна / Месяц / Оракул
Что делал: [описание действия]
Что произошло: [описание ошибки или что показалось непонятным]
```
