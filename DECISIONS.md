## [2026-03-2026-04] — Mini App: GitHub Pages (автономный SPA)

**Решение:** Mini App (tg-app/) деплоится на GitHub Pages как автономный SPA. Не делает API-вызовов — все данные в data.js. Бот ↔ API (Beget) разделены.

**Причина:** GitHub Pages = HTTPS бесплатно, CDN, без серверных расходов. Автономность SPA = скорость загрузки, работа офлайн.

**Ограничение:** при персонализированных данных нужен API-вызов через HMAC-аутентификацию (уже реализовано в api.py).

**Не пересматривать**
