# Как запустить Selenyx

## 1. Получи токен у BotFather (уже сделано ✅)

## 2. Вставь токен в файл `.env`

Открой файл `.env` и убедись, что там есть строка:
```
BOT_TOKEN=твой_токен_от_BotFather
```

## 3. Запусти бота

Открой терминал в папке проекта и введи:

```bash
pkill -9 -f "bot.py" 2>/dev/null; sleep 6
venv/bin/python3 bot.py > bot.log 2>&1 &
sleep 8 && cat bot.log
```

Если видишь **«Бот запущен»** — всё работает.
Открой Telegram, найди своего бота и напиши `/start`.

## 4. Остановить бота

```bash
pkill -9 -f "bot.py"
```

## 5. Посмотреть логи

```bash
cat bot.log
```
