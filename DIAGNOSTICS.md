# DIAGNOSTICS.md — Психологическая диагностика в Selenyx

> Создано: 2026-04-01
> Статус: концепция зафиксирована, реализация — следующий спринт

---

## Суть идеи

Ежедневная короткая проективная диагностика — пользователь не отвечает на вопросы,
а реагирует инстинктивно на образы/цвета/символы. Как тест Люшера, но компактный (10–15 сек).

**Принцип:** нет правильных ответов. Только внутренняя реакция.

---

## Пользовательский сценарий (флоу)

```
Вкладка «День»
    ↓
Карточка-приглашение: «Диагностика дня — 10 сек»
    ↓
Карусель превью (горизонтальный скролл, 3–4 типа на выбор)
  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐
  │  🎨 Цвета  │  │  🌊 Стихии │  │  🌙 Символы│
  │  10 сек    │  │  15 сек    │  │  10 сек    │
  └─────────────┘  └─────────────┘  └─────────────┘
    ↓ (пользователь выбирает)
Сам тест — визуальный выбор (2–3 клика)
    ↓
Трактовка результата — короткий психологический инсайт
    ↓
Кнопки «🎯 В точку» / «↩ Не про меня»
```

---

## Типы диагностик (банк 20–30 штук)

| Тип | Описание | Время |
|-----|----------|-------|
| 🎨 Цвета | 6 цветовых карточек → выбери 2 которые притягивают сейчас | 10 сек |
| 🌊 Стихии | Огонь / Вода / Земля / Воздух → что откликается | 5 сек |
| 🌙 Символы | Луна / Зеркало / Ключ / Птица / Огонь / Камень → выбери один | 10 сек |
| 🏔 Пространство | Лес / Океан / Горы / Город → где ты сейчас внутри | 5 сек |
| 🌅 Время суток | Рассвет / Полдень / Закат / Ночь → что резонирует | 5 сек |
| 🪵 Фактуры | Шёлк / Дерево / Металл / Земля → что хочется почувствовать | 5 сек |

30 диагностик в банке = пользователь месяц видит новое каждый день. Повтор не ощущается.

---

## Ротация контента

- Без привязки к астро-контексту — чисто психологические
- Каждый день — следующая в банке (по индексу дня года % количество_тестов)
- Пользователь видит карусель и может выбрать другой тип если не откликается текущий
- Контент пишем один раз: трактовки для каждого варианта каждой диагностики

---

## Место в интерфейсе

- **Вкладка «День»** — карточка-приглашение внизу (после доменов)
- **Онбординг** — один вводный слайд при первом входе (знакомство с форматом)
- После прохождения кнопки **«🎯 В точку» / «↩ Не про меня»** — оценка точности трактовки

---

## Контент — что нужно написать

Для каждого теста в банке:
- Название и тип
- Набор визуальных вариантов (4–6 штук)
- Трактовка для каждого варианта (2–4 предложения)
- Тон: психологический, без осуждения, поддерживающий

---

## Связь с кнопками «В точку / Не про меня»

Кнопки появляются ТОЛЬКО после прочтения трактовки диагностики.
Это осмысленный фидбек — пользователь оценивает точность психологического портрета,
а не астрологического прогноза (как было раньше).

Данные фидбека → в будущем помогут персонализировать подбор диагностик.
