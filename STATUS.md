## ⏳ Следующий шаг — Шаг 7: Закрытый тест

**Что сделать до теста:**
1. Убедиться что DEMO_MODE=false на Beget:
   ```bash
   ssh root@45.9.43.149 "grep DEMO_MODE /home/selenyx/app/.env"
   ```
2. Обновить BETA_TEST.md (готов — см. файл)
3. Пригласить 10–15 человек: подруги, знакомые, астро-чаты
4. Через 3–5 дней: retention в SQLite
5. Через неделю: 3 вопроса (понравилось / непонятно / вернётся?)

**Гипотезы:**
- 30%+ вернутся на день 3
- 50%+ откроют оракул
- 20%+ включат уведомления
